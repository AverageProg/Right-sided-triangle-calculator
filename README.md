# Right-sided-triangle-calculator
Interactive web tool for solving right-angled triangles. Detects user inputs to calculate hypotenuse, side lengths, or corner degrees using inverse trigonometric functions. Built with HTML ,CSS and JavaScript.

## ✨ Features
* **Smart Input Detection:** Uses `if/else` logic to determine what you're trying to solve based on the fields you fill in.
* **Trigonometry Engine:** Calculates side lengths and angles using Sin, Cos, and Tan.
* **Pythagorean Solver:** Automatically finds the hypotenuse ($a^2 + b^2 = c^2$) or missing legs.
* **Degree/Radiant Handling:** Handles all the complex background math so you can work in simple degrees.
* **School-Ready Results:** Provides rounded whole-number results perfect for STEM assignments and quick verification.

## 🚀 How to Use
1.  **Solve by Sides:** Enter any two sides (a, b, or c) to find the third side and all internal angles.
2.  **Solve by Trig:** Enter an angle (degrees) and one known side to find the remaining dimensions.
3.  **Reset:** Use the built-in reset button to clear all fields and calculations instantly for a new problem.

## 🛠️ Technical Stack
* **HTML5** - Semantic structure for inputs.
* **CSS3** - High-visibility neon UI design.
* **JavaScript** - Custom math engine using the `Math` library.

## 📽️ Project Background
This project was built to help STEM students simplify triangle geometry and verify homework solutions quickly. It focuses on clean code, accurate math, and an intuitive user experience.
